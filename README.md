# Netflix Dashboard Project

##  Overview
This is an **interactive Power BI dashboard** analyzing the Netflix dataset.  
It provides insights into Netflix’s content library, including **total titles, genre distribution, release trends over time, and global distribution by country**.

**Key Features:**
- Count of total Netflix titles (Movies & TV Shows)  
- Breakdown of content by **type** (Movies vs TV Shows)  
- Top 10 genres by number of titles  
- Historical release trends over time  
- Map visualization showing **number of titles per country**

---

##  Visuals
| Visual | Description |
|-------|-------------|
| **Card** | Shows total number of Netflix titles |
| **Pie Chart** | Distribution of Movies vs TV Shows |
| **Bar Chart** | Top 10 genres |
| **Line Chart** | Releases over time |
| **Map** | Titles by country |

---

##  Tools & Dataset
- **Tool:** Power BI Desktop  
- **Dataset:** [Netflix Movies and TV Shows](https://www.kaggle.com/datasets/shivamb/netflix-shows) (CSV)

---

##  How to Use
1. Open the `.pbix` file in Power BI Desktop.  
2. Explore visuals interactively: hover over charts for tooltips, filter by type, country, or genre.  
3. Modify or add additional insights as needed.

##  Insights You Can Highlight
- Netflix has a diverse range of content, with **Movies dominating slightly over TV Shows**.  
- Certain genres like **Dramas and Comedies** dominate the top 10 list.  
- Content release trends show **increasing number of titles over time**.  
- **USA and India** are among the countries with the highest number of titles.
